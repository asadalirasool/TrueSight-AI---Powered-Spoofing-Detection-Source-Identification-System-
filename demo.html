⚙️ Deployment Instructions:

To run the complete system:

  1. Install dependencies: pip install -r backend/requirements.txt
  2. Navigate to backend: cd backend
  3. Start server: python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
  4. Access API at: http://localhost:8000

🌐 API Endpoints:

GET / - Root endpoint
GET /health - Health check
POST /api/v1/detection/detect-deepfake - Main detection
POST /api/v1/detection/detect-video-deepfake - Video-only detection
POST /api/v1/detection/detect-audio-deepfake - Audio-only detection
POST /api/v1/detection/detect-face - Face detection with margin
